# List of Products Screen

It displays a list of products for an online store application. Users can browse available products and view basic details.

## 📂 Features

- Browse all products
- Add product to wishlist.
- Add product to cart

## 🛠 Tech Stack

- Flutter & Dart
- Cubit (State management)
- DIO
- MVVM Architecture
- Dummyjson REST API
